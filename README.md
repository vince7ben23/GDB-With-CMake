# GDB with CMake
- A toy project to integrate GDB with CMake.

## How to Build
```bash
# build
./build.sh

# build with debug mode
./build.sh -g
```

## Third-party Libs
### Install spdlog

```bash
mkdir -p src/spdlog
```
Header-only version
- Copy [the spdlog folder](https://github.com/gabime/spdlog/tree/v1.x/include/spdlog) to src/spdlog
